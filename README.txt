River Inversion 2D
==================

The files in this repository are a set of python scripts which implement the 
river profile to uplift history inversion alogrithm described in Roberts, Rudge,
Richardson and White (2014) [1].

Installation
============
In order to run these scripts, you need to have the following installed:

* python 2.7
* numpy (version or later)
* dolfin 1.3.0 (www.fenicsproject.org)
* sklearn (version)
